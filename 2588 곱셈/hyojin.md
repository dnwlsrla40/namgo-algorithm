## 2588번 곱셈
### 곱셈
문제 출처 : https://www.acmicpc.net/problem/2588

## 내가 푼 코드

```
A = 473
B = 3856

C = str(B)
D = len(C)


def add(A, B):
    for i in range(int(D), 0, -1):
        B_ = C[i - 1]
        print(A * int(B_))
    print(A * B)
```

add(A, B)

## 생각한 방향
두 번째로 주어진 숫자의 각 자리 수를 구해 첫 번째 수와 곱한 결과를 출력.
(3),(4),(5)는 곱셈된 숫자만 구하면 되지만, 총합은 거기에 각 인자 번호 만큼 10을 곱해주어야 재대로 나옴
